# FlowSense AI

## AI Powered Process Mining Platform

FlowSense AI is a MERN Stack based web application that helps organizations analyze business processes using AI Process Mining.

## Features

- User Registration
- User Login
- Home Page
- Product Page
- Solutions Page
- Features Page
- Industries Page
- Pricing Page
- About Page
- FAQ Page
- Contact Page
- Dashboard
- Upload CSV Event Log
- Process Health Analysis
- Health Score
- AI Recommendations

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- JavaScript
- HTML
- CSS
